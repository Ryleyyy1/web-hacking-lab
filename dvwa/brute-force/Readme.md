# DVWA – Brute Force Authentication

This repository documents a brute force attack conducted against
the Damn Vulnerable Web Application (DVWA) in a controlled local
environment for educational purposes.

⚠️ All tests were performed on localhost.  
⚠️ No real systems were harmed.

## Target
- Application: Damn Vulnerable Web Application (DVWA)
- Module: Brute Force
- Security Level: Low

## Objective
Understand how weak authentication mechanisms can be exploited
using both manual testing and automated scripting techniques.
